# Product Selection Rules

## PLC Selection Rules

Required information before recommending PLC:

- I/O points
- Digital input count
- Digital output count
- Output type: relay or transistor
- Power supply
- Communication interface
- Protocol
- Application
- Quantity

If output type is missing, ask whether relay output or transistor output is required.

## VFD Selection Rules

Required information before recommending VFD:

- Motor power in kW
- Input voltage
- Phase
- Application
- Communication requirement
- Motor type
- Quantity

If motor power or voltage is missing, ask clarification before recommending exact model.

## HMI Selection Rules

Required information before recommending HMI:

- Screen size
- Resolution
- Communication interface
- Protocol
- PLC compatibility
- Power supply
- Quantity

For Modbus TCP, recommend Ethernet HMI models.

## Industrial Switch Selection Rules

Required information before recommending industrial switch:

- Port count
- Speed
- Managed or unmanaged type
- PoE requirement
- Fiber port requirement
- Power input
- Temperature range
- Quantity

For factory backbone or redundant network, managed switches are preferred.
