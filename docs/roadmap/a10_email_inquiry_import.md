# A10 邮件询盘导入增强 Email Inquiry Import Enhancement

## 1. 阶段目标

A10 目标是规划邮件询盘导入增强，让外贸场景更贴近真实工作流。

本阶段只做路线图和技术设计，不开发代码。

## 2. 计划能力

邮件导入字段：

- 邮件标题 subject。
- 发件人 sender。
- 邮件正文 body。
- 客户公司 company。
- 国家 country。
- 联系方式 contact information。
- 附件 metadata。

Agent 输出增强：

- 识别客户公司和国家。
- 抽取联系方式。
- 生成英文回复草稿。
- 支持复制回复草稿。
- 支持导出 Markdown Demo Report。

## 3. 用户流程

```text
业务员粘贴邮件内容
-> 系统解析基础字段
-> Agent 分析询盘
-> 展示候选产品 / 缺失信息 / 风险提示
-> 业务员编辑英文回复草稿
-> 人工审核记录
```

## 4. 暂不做

- 不自动登录邮箱。
- 不自动发送邮件。
- 不连接真实企业邮箱。
- 不处理复杂附件 OCR。
- 不做邮件线程同步。
- 不承诺库存、交期或报价。

## 5. 后续验收标准

- 邮件字段解析逻辑有测试。
- Email Inquiry 页面或入口不影响 Website Inquiry。
- 回复草稿仍必须人工审核。
